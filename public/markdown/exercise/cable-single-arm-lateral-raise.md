# Cable Single Arm Lateral Raise - Forge Exercise Library

## Quick Answer

The cable one arm lateral raise is an isolation exercise targeting the deltoid muscles, particularly the lateral (side) head. It is performed using a cable machine, which provides constant tension throughout the movement. This exercise helps build shoulder width and strength.

**Details:**
- **Body Part:** Shoulders
- **Equipment:** Cable
- **Target Muscle:** Shoulders
- **Secondary Muscles:** Traps, Triceps
- **Difficulty:** Intermediate
- **Category:** Strength

---

## How to Perform Cable Single Arm Lateral Raise

1. Stand with your feet shoulder-width apart, facing the cable machine.
2. Hold the cable handle with one hand, palm facing down, and stand far enough away from the machine so that there is tension on the cable.
3. Keep your arm straight and slowly raise it out to the side until it is parallel to the ground.
4. Pause for a moment at the top, then slowly lower your arm back down to the starting position.
5. Repeat for the desired number of repetitions, then switch sides.
