# EZ Bar Curl vs Barbell Curl

## Quick answer

EZ Bar Curl targets the biceps using ez bar (Beginner), while Barbell Curl targets the biceps using barbell (Beginner).

## FAQ

### What's the difference between EZ Bar Curl and Barbell Curl?

EZ Bar Curl targets the biceps using ez bar, while Barbell Curl targets the biceps using barbell.

EZ Bar Curl is beginner-friendly.
