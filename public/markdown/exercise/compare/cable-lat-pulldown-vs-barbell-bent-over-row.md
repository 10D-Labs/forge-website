# Cable Lat Pulldown vs Barbell Bent Over Row

## Quick answer

Cable Lat Pulldown targets the lats using cable (Beginner), while Barbell Bent Over Row targets the upper back using barbell (Intermediate).

---

## Comparison

| Attribute | Cable Lat Pulldown | Barbell Bent Over Row |
|-----------|--------------------|-----------------------|
| Body Part | Back | Back |
| Target | Lats | Upper Back |
| Equipment | Cable | Barbell |
| Difficulty | Beginner | Intermediate |
| Category | Strength | Strength |
| Secondary | Biceps, Rhomboids, Rear Deltoids | Biceps, Forearms |

## FAQ

### What's the difference between Cable Lat Pulldown and Barbell Bent Over Row?

Cable Lat Pulldown targets the lats using cable, while Barbell Bent Over Row targets the upper back using barbell.

### Which is better for beginners?

Cable Lat Pulldown is beginner-friendly.

### Can I do both in the same workout?

Yes, you can include both exercises in your routine since they both target back. Alternate between them to work your muscles from different angles.
