# Short Stride Run (Time) - Forge Exercise Library

## Quick Answer

Short stride run is a cardiovascular exercise performed by running with quick, short steps. It primarily targets the cardiovascular system and also engages the quadriceps, hamstrings, and calves. The shorter stride length increases cadence and can help improve running efficiency.

**Details:**
- **Body Part:** Cardio
- **Equipment:** Body Weight
- **Target Muscle:** Cardiovascular
- **Secondary Muscles:** Quadriceps, Hamstrings, Calves
- **Difficulty:** Beginner
- **Category:** Cardio
- **Tracking Type:** time

---

## How to Perform Short Stride Run (Time)

1. Find an open space to perform the exercise.
2. Stand tall with your feet hip-width apart.
3. Start running with quick, short steps, maintaining rapid leg turnover.
4. Keep your feet close to the ground and avoid overstriding.
5. Continue running with short strides for your target duration.
